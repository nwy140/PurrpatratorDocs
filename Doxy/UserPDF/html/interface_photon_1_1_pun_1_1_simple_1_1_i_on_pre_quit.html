void OnPreQuit ()
 

Detailed Description

Callback Interface for OnPreQuit().

Triggered when NetMaster OnApplicationQuit() is called. NetMaster's script execution order is set to fire prior to all other MonoB scripts, so OnPreQuit() will fire prior to OnApplicationQuit() on all scripts.
