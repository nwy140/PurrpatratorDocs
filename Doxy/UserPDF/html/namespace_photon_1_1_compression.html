enum Photon.Compression.IndicatorBit

This enum indicates if an extra bit should be added to indicate cases of Zero value.

When value is zero, only the one bit is sent. Useful for values that spend more time at zero than not, and delta frames are not an option.

◆ IndicatorBits

enum Photon.Compression.IndicatorBits

This enum indicates if an extra bit should be added to indicate cases of Zero value, XXXXX preValue, or two bits should be added to indicate Zero, Min, Max.

When value is one of those, only the indicator bits will be written. Use this if these values are expected to spend most of their life at zero, min or max.

◆ SyncAs

enum Photon.Compression.SyncAs

Indicates how a [Pack] syncvar acts.

Enumerator
Auto 

For PackObjects this uses the default of State.

For Pack syncvar fields this indicates they should use the PackObject setting.

State 

Pack syncvar replicates changes in value to other clients, and stays at that value until changed by the owner.

Trigger 

Pack syncvar resets value on owner after its value is captured/sent.

Reset is done with value = new T(). Only works on struct types.
