\doxysubsection{Detailed Description}
This simple \doxyref{Chat}{p.}{namespace_photon_1_1_chat} UI demonstrate basics usages of the \doxyref{Chat}{p.}{namespace_photon_1_1_chat} Api 

The \doxyref{Chat\+Client}{p.}{class_photon_1_1_chat_1_1_chat_client} basically lets you create any number of channels.

some friends are already set in the \doxyref{Chat}{p.}{namespace_photon_1_1_chat} demo \char`\"{}\+Demo\+Chat-\/\+Scene\char`\"{}, \textquotesingle{}Joe\textquotesingle{}, \textquotesingle{}Jane\textquotesingle{} and \textquotesingle{}Bob\textquotesingle{}, simply log with them so that you can see the status changes in the Interface

Workflow\+: Create \doxyref{Chat\+Client}{p.}{class_photon_1_1_chat_1_1_chat_client}, Connect to a server with your App\+ID, Authenticate the user (apply a unique name,) and subscribe to some channels. Subscribe a channel before you publish to that channel!

Note\+: Don\textquotesingle{}t forget to call \doxyref{Chat\+Client.\+Service()}{p.}{class_photon_1_1_chat_1_1_chat_client_aa08129ee3ee957c825e0a91a61e97993} on Update to keep the Chatclient operational.
