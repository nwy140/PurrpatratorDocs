\doxysubsection{Detailed Description}
The default implementation of a Prefab\+Pool for PUN, which actually Instantiates and Destroys Game\+Objects but pools a resource. 

This pool is not actually storing Game\+Objects for later reuse. Instead, it\textquotesingle{}s destroying used Game\+Objects. However, prefabs will be loaded from a Resources folder and cached, which speeds up Instantiation a bit.

The Resource\+Cache is public, so it can be filled without relying on the Resources folders.
