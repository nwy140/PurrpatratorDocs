Called when the server sent the response to a Find\+Friends request. 

After calling Op\+Find\+Friends, the Master Server will cache the friend list and send updates to the friend list. The friends includes the name, user\+Id, online state and the room (if any) for each requested user/friend.

Use the friend\+List to update your UI and store it, if the UI should highlight changes.
