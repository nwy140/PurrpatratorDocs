\doxysubsection{Detailed Description}
Use this in scenes you want to leave Control for connection and pun related commands to \doxyref{Cockpit}{p.}{namespace_photon_1_1_pun_1_1_demo_1_1_cockpit}. 

It requires Connect\+And\+Join\+Random, which it will control for connecting should the \doxyref{Cockpit}{p.}{namespace_photon_1_1_pun_1_1_demo_1_1_cockpit} scene not be present or succesfully loaded.
