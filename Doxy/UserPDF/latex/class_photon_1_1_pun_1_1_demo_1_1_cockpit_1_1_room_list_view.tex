Called for any update of the room-\/listing while in a lobby (In\+Lobby) on the Master Server. 

Each item is a Room\+Info which might include custom properties (provided you defined those as lobby-\/listed when creating a room). Not all types of lobbies provide a listing of rooms to the client. Some are silent and specialized for server-\/side matchmaking.
