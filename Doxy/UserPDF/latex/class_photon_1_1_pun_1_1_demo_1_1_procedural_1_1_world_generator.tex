\doxysubsection{Detailed Description}
The World Generator creates a world based on four options the current Master\+Client can set. 

These options are available on the Ingame Control Panel. If those options are confirmed by the current Master\+Client, they will be stored in the Custom Room Properties to make them available on all clients. These options are\+: 1) a numerical Seed to make sure that each client generates the same world and to avoid Random functions and \textquotesingle{}network-\/instantiate\textquotesingle{} everything 2) the World Size to describe how large the generated world should be 3) the \doxyref{Cluster}{p.}{class_photon_1_1_pun_1_1_demo_1_1_procedural_1_1_cluster} Size to describe how many Blocks are inside each \doxyref{Cluster}{p.}{class_photon_1_1_pun_1_1_demo_1_1_procedural_1_1_cluster} 4) the World Type to make the generated world appear in different \textquotesingle{}looks\textquotesingle{}.
