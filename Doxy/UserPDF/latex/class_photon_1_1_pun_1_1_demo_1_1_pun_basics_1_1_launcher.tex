Called when entering a room (by creating or joining it). 

Called on all clients (including the Master Client). 

This method is commonly used to instantiate player characters. If a match has to be started \char`\"{}actively\char`\"{}, you can call an \doxyref{Pun\+RPC}{p.}{class_photon_1_1_pun_1_1_photon_view_a92bc79e36a53f2137e4b45e5162bdd7b} triggered by a user\textquotesingle{}s button-\/press or a timer.

When this is called, you can usually already access the existing players in the room via \doxyref{Photon\+Network.\+Player\+List}{p.}{class_photon_1_1_pun_1_1_photon_network_a13cd7f74a3672513b43bf224968eb9de}. Also, all custom properties should be already available as Room.\+custom\+Properties. Check Room..Player\+Count to find out if enough players are in the room to start playing.
