\doxysubsection{Detailed Description}
This container is used in On\+Photon\+Serialize\+View() to either provide incoming data of a \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view} or for you to provide it. 

The Is\+Writing property will be true if this client is the \char`\"{}owner\char`\"{} of the \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view} (and thus the Game\+Object). Add data to the stream and it\textquotesingle{}s sent via the server to the other players in a room. On the receiving side, Is\+Writing is false and the data should be read.

Send as few data as possible to keep connection quality up. An empty \doxyref{Photon\+Stream}{p.}{class_photon_1_1_pun_1_1_photon_stream} will not be sent.

Use either \doxyref{Serialize()}{p.}{class_photon_1_1_pun_1_1_photon_stream_a5bd4085dcff91fd8f08ce7298adc4aca} for reading and writing or \doxyref{Send\+Next()}{p.}{class_photon_1_1_pun_1_1_photon_stream_adf067be77d2034b07fd696d97110a57a} and \doxyref{Receive\+Next()}{p.}{class_photon_1_1_pun_1_1_photon_stream_a50038b09fdec1b4d1072c9f8b5a231dc}. The latter two are just explicit read and write methods but do about the same work as \doxyref{Serialize()}{p.}{class_photon_1_1_pun_1_1_photon_stream_a5bd4085dcff91fd8f08ce7298adc4aca}. It\textquotesingle{}s a matter of preference which methods you use.
