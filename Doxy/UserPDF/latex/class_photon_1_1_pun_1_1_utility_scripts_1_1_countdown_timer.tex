\doxysubsection{Detailed Description}
This is a basic, network-\/synced \doxyref{Countdown\+Timer}{p.}{class_photon_1_1_pun_1_1_utility_scripts_1_1_countdown_timer} based on properties. 

In order to start the timer, the Master\+Client can call \doxyref{Set\+Start\+Time()}{p.}{class_photon_1_1_pun_1_1_utility_scripts_1_1_countdown_timer_a402d0ea749a694f65303b1bc82c376a1} to set the timestamp for the start. The property \textquotesingle{}Start\+Time\textquotesingle{} then contains the server timestamp when the timer has been started.

In order to subscribe to the Countdown\+Timer\+Has\+Expired event you can call \doxyref{Countdown\+Timer.\+On\+Countdown\+Timer\+Has\+Expired}{p.}{class_photon_1_1_pun_1_1_utility_scripts_1_1_countdown_timer_a872822dedbe8007e7c978868b3f5b30d} += On\+Countdown\+Timer\+Is\+Expired; from Unity\textquotesingle{}s On\+Enable function for example. For unsubscribing simply call \doxyref{Countdown\+Timer.\+On\+Countdown\+Timer\+Has\+Expired}{p.}{class_photon_1_1_pun_1_1_utility_scripts_1_1_countdown_timer_a872822dedbe8007e7c978868b3f5b30d} -\/= On\+Countdown\+Timer\+Is\+Expired;.

You can do this from Unity\textquotesingle{}s On\+Enable and On\+Disable functions.
