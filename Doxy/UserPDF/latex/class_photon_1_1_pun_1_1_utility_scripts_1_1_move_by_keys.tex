\doxysubsection{Detailed Description}
Very basic component to move a Game\+Object by WASD and Space. 

Requires a \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view}. Disables itself on Game\+Objects that are not owned on Start.

Speed affects movement-\/speed. Jump\+Force defines how high the object \char`\"{}jumps\char`\"{}. Jump\+Timeout defines after how many seconds you can jump again.
