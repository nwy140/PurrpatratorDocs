\doxysubsection{Detailed Description}
Basic GUI to show traffic and health statistics of the connection to \doxyref{Photon}{p.}{namespace_photon}, toggled by shift+tab. 

The shown health values can help identify problems with connection losses or performance. \doxyref{Example}{p.}{namespace_photon_1_1_pun_1_1_example}\+: If the time delta between two consecutive Send\+Outgoing\+Commands calls is a second or more, chances rise for a disconnect being caused by this (because acknowledgements to the server need to be sent in due time).
