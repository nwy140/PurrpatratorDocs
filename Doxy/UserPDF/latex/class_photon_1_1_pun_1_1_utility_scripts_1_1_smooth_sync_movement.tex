{\footnotesize\ttfamily void Photon.\+Pun.\+Utility\+Scripts.\+Smooth\+Sync\+Movement.\+On\+Photon\+Serialize\+View (\begin{DoxyParamCaption}\item[{\textbf{ Photon\+Stream}}]{stream,  }\item[{\textbf{ Photon\+Message\+Info}}]{info }\end{DoxyParamCaption})}



Called by PUN several times per second, so that your script can write and read synchronization data for the \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view}. 

This method will be called in scripts that are assigned as Observed component of a \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view}.~\newline
 \doxyref{Photon\+Network.\+Serialization\+Rate}{p.}{class_photon_1_1_pun_1_1_photon_network_add2c5cc89d654a90db576bc9a6e0411f} affects how often this method is called.~\newline
 \doxyref{Photon\+Network.\+Send\+Rate}{p.}{class_photon_1_1_pun_1_1_photon_network_a84b1a2edd7844e3f42b1e58e6b4fc9c8} affects how often packages are sent by this client.~\newline


Implementing this method, you can customize which data a \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view} regularly synchronizes. Your code defines what is being sent (content) and how your data is used by receiving clients.

Unlike other callbacks, {\itshape On\+Photon\+Serialize\+View only gets called when it is assigned to a \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view}} as Photon\+View.\+observed script.

To make use of this method, the \doxyref{Photon\+Stream}{p.}{class_photon_1_1_pun_1_1_photon_stream} is essential. It will be in \char`\"{}writing\char`\"{} mode\char`\"{} on the client that controls a Photon\+View (\+Photon\+Stream.\+Is\+Writing == true) and in \char`\"{}reading mode\char`\"{} on the remote clients that just receive that the controlling client sends.  \+If you skip writing any value into the stream, PUN will skip the update. Used carefully, this can conserve bandwidth and messages (which have a limit per room/second).  \+Note that On\+Photon\+Serialize\+View is not called on remote clients when the sender does not send any update. This can\textquotesingle{}t be used as \char`\"{}x-\/times per second \doxyref{Update()}{p.}{class_photon_1_1_pun_1_1_utility_scripts_1_1_smooth_sync_movement_a44dcd0f5540e1fb1c6fc2bcee1a8f059}"{}.
