\doxysubsubsection{BestRegionSummaryFromStorage}
{\footnotesize\ttfamily string Photon.\+Realtime.\+App\+Settings.\+Best\+Region\+Summary\+From\+Storage}



Set to a previous Best\+Region\+Summary value before connecting. 

This is a value used when the client connects to the \char`\"{}\+Best Region\char`\"{}. If this is null or empty, all regions gets pinged. Providing a previous summary on connect, speeds up best region selection and makes the previously selected region \char`\"{}sticky\char`\"{}.

Unity clients should store the Best\+Region\+Summary in the Player\+Prefs. You can store the new result by implementing \doxyref{IConnection\+Callbacks.\+On\+Connected\+To\+Master}{p.}{interface_photon_1_1_realtime_1_1_i_connection_callbacks_a52dfd77079d550168c7ff7aa0ce01a6c}. If \doxyref{Load\+Balancing\+Client.\+Summary\+To\+Cache}{p.}{class_photon_1_1_realtime_1_1_load_balancing_client_a0e2dcf5f096158928ad076e30685461d} is not null, store this string. To avoid storing the value multiple times, you could set Summary\+To\+Cache to null.
