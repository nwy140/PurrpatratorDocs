\doxysubsubsection{DisconnectAfterKeepAlive}
{\footnotesize\ttfamily bool Photon.\+Realtime.\+Connection\+Handler.\+Disconnect\+After\+Keep\+Alive = false}



Option to let the fallback thread call Disconnect after the Keep\+Alive\+In\+Background time. 

Default\+: false.

If set to true, the thread will disconnect the client regularly, should the client not call Send\+Outgoing\+Commands / Service. This may happen due to an app being in background (and not getting a lot of CPU time) or when loading assets.

If false, a regular timeout time will have to pass (on top) to time out the client.
