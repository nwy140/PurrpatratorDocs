Operation to handle this client\textquotesingle{}s interest groups (for events in room). 

Note the difference between passing null and byte[0]\+: null won\textquotesingle{}t add/remove any groups. byte[0] will add/remove all (existing) groups. First, removing groups is executed. This way, you could leave all groups and join only the ones provided.

Changes become active not immediately but when the server executes this operation (approximately RTT/2).
