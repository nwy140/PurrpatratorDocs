\doxysubsection{Detailed Description}
Refers to a specific lobby on the server. 

Name and Type combined are the unique identifier for a lobby.~\newline
 The server will create lobbies \char`\"{}on demand\char`\"{}, so no registration or setup is required.~\newline
 An empty or null Name always points to the \char`\"{}default lobby\char`\"{} as special case.
