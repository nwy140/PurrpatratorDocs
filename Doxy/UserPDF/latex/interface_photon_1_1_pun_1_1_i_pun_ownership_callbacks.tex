\doxysubsection{Detailed Description}
Global Callback interface for ownership changes. 

These callbacks will fire for changes to ANY \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view} that changes. Consider using \doxyref{IOn\+Photon\+View\+Controller\+Change}{p.}{interface_photon_1_1_pun_1_1_i_on_photon_view_controller_change} for callbacks from a specific \doxyref{Photon\+View}{p.}{class_photon_1_1_pun_1_1_photon_view}.
