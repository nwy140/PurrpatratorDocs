\doxysubsection{Detailed Description}
These callbacks are called by \doxyref{Net\+Master}{p.}{class_photon_1_1_pun_1_1_simple_1_1_net_master} to serialize data to the tick stream after the Frame\+Id value, and before all \doxyref{Net\+Object}{p.}{class_photon_1_1_pun_1_1_simple_1_1_net_object} data is written. 

The primary use case for this would be serialization of user inputs, but can be used for any other global, non net entity specific serialization, such as scores, timers, global events, etc.
