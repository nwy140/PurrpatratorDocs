Detailed Description

The World Generator creates a world based on four options the current MasterClient can set.

These options are available on the Ingame Control Panel. If those options are confirmed by the current MasterClient, they will be stored in the Custom Room Properties to make them available on all clients. These options are: 1) a numerical Seed to make sure that each client generates the same world and to avoid Random functions and 'network-instantiate' everything 2) the World Size to describe how large the generated world should be 3) the Cluster Size to describe how many Blocks are inside each Cluster 4) the World Type to make the generated world appear in different 'looks'.
